# SportTrackerFresh Design Brief

## Visual Direction

Dark-first athletic mobile UI with premium spacing, rounded touch-friendly controls, strong metric cards, and restrained accents.

## Core Mood

- focused
- modern
- calm
- data-driven
- premium

## Color Tokens

- background: `#0B0D10`
- surface: `#13171C`
- surface-raised: `#1A1F26`
- text-primary: `#F5F7FA`
- text-secondary: `#B7C0CC`
- text-muted: `#7E8794`
- accent-green: `#22C55E`
- accent-violet: `#7C8CFF`
- danger: `#F87171`
- warning: `#F59E0B`

## UI Rules

- Prefer single-column layouts on phones
- Keep primary actions large and reachable
- Make metrics visually dominant
- Use subtle borders instead of heavy shadows
- Avoid noisy gradients and overdecorated cards
