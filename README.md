# SportTrackerFresh

Новый старт проекта в спортивной тематике на:

- Expo
- React Native
- TypeScript
- SQLite
- Zustand
- React Navigation
- React Hook Form
- Zod
- Victory Native

## Что уже подготовлено

- базовая архитектура `src/`
- нижняя навигация
- темная спортивная тема
- стартовые экраны Dashboard / Workouts / Profile
- Zustand store с мок-данными
- каркас для SQLite

## Что нужно сделать после установки Node.js

1. Установить Node.js LTS
2. Выполнить `npm install`
3. Выполнить `npx expo start`

## Следующий шаг

После установки зависимостей можно сразу переходить к:

- CRUD тренировок
- локальной БД SQLite
- форме создания упражнения
- графикам прогресса
